Ко всему этому Там же лежит моё веб-приложение, которое отображает время. Так же прочёл паттерны 12 факторов, которые разработали в компании Netflix, 
на которые была ссылка в книге (https://12factor.net/ru/) 

Глава 3 - Внутреннее устройство и возможности Spring Boot.

Данная глава показывает различные возможности Spring Boot и то, какими способами можно сделать ту или иную вещь (Запуск, создание кастомного баннера во время запуска, и отключение его вовсе, подключение прослушивателей, профилей, логгирования, соединение классов, бинов, создание кастомных профилей и кастомных значений в них)

Глава 4 - Создание веб-приложений.

Данная глава показывает как делать веб приложение (сервер) и клинтское приложение (клиент). За основу взят пример из первой главы, но уже модифицированный. Можно добавлять, удалять, редактировать, ставить значение "выполнено" задачам, которые создаёшь сам. Что касается клиентского приложения, то оно автоматически создаёт задачу "Drink some water", редактирует его, делает выполненым и наконец удаляет. Так же вместо curl использовал Postman, т.к. он болле дружественно настроен к windows, по сравнению с curl и его экранированием ковычек
